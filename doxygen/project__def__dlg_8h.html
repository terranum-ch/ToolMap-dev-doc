Detailed Description

Definition of the Project definition dialog.

This file provides the class needed for dealing with project definition (layers, object and optional field). This dialog is called either for a new project or changing the parameters of an existing one. The two class implemented are : ProjectDefDLG for the dialog and ProjectDefList for the list inside the dialog.
